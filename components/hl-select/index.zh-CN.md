基于 HankLiu 内部需求封装的 Select 组件，在 Antd Select 组件基础上添加了 lockHeight 参数，其余参数与用法参考 [Select](/components/select/) 组件。

## API

### Select props

| 参数       | 说明           | 类型    | 默认值 | 版本 |
| ---------- | -------------- | ------- | ------ | ---- |
| lockHeight | 固定输入框高度 | boolean | false  |      |
